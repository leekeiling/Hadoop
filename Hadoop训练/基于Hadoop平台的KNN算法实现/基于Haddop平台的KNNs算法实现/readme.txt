KNN

本程序通过JAVA语言实现KNN算法，并Hadoop上实现分布式计算。




包括的文件：
Knn.java			-------------实现Knn算法
KnnInstance.java		-------------用于处理训练文本和测试文本，辅助实现Knn算法
RESULT				-------------测试集结果





训练集文本：		       ------共623行，每行的最后一数字为对应行的标签，标签分为-1，+1，其余为对应属性，共2697个。
测试集样本		       ------共311行，属性2697个	      
测试输出结果文件：	       ------共311行，两列，第一列为测试集索引，第二列为预测标签
